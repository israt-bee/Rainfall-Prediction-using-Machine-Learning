# Rainfall-Prediction-using-Machine-Learning

## Overview
This project uses machine learning to predict rainfall patterns.

## Features
- Data preprocessing and analysis
- Multiple ML models implementation
- Performance evaluation and comparison

## Getting Started
### Prerequisites
- Python 3.x
- Required libraries (see requirements.txt)

### Installation
```bash
pip install -r requirements.txt
```

### Usage
Run the main prediction script:
```bash
python predict_rainfall.py
```

## Models
- Linear Regression
- Random Forest
- Neural Networks

## Results
📈 Model Performance:
Model	Accuracy
Logistic Regression	82.4%
Decision Tree	78.1%
Random Forest	87.6% 
##  Best Performing Model
🏆 Best Performing Model:
👉 Random Forest Classifier achieved the highest accuracy of 87.6%, making it the most effective model for rainfall prediction in this project.
